ภาพรวม — ไฟล์ที่ต้องการ

เว็บ INeedPOSN ประกอบด้วยไฟล์ HTML ทั้งหมด ไม่ต้องใช้ server ใดๆ เพราะใช้ Google Apps Script เป็น backend ฟรี

📁 ineedposn/ (โฟลเดอร์ที่จะ upload) ├── index.html       ← หน้าหลัก + สมัครเรียน ├── learn.html       ← ห้องเรียนนักเรียน ├── admin.html       ← หลังบ้าน Admin ├── qr-promptpay.png  ← QR PromptPay ของคุณ ├── qr-discord.png    ← QR เข้า Discord ├── photo-bio.jpg     ← รูปพี่พร้อม ├── photo-chem.jpg    ← รูปพี่หมีลี่ └── photo-com.jpg     ← รูปพี่น้ำแข็ง
⚠️ ข้อสำคัญ — เรื่อง localStorage

เว็บนี้ใช้ localStorage เก็บข้อมูล ซึ่งหมายความว่าข้อมูลถูกเก็บไว้ในเบราว์เซอร์แต่ละเครื่อง

ปัญหา: ถ้านักเรียน A สมัครจากโทรศัพท์ → ข้อมูลอยู่ใน localStorage ของโทรศัพท์เขา แต่เมื่อพี่พร้อมเปิด admin.html จากคอมพิวเตอร์อีกเครื่อง → จะไม่เห็นออเดอร์นั้น

วิธีแก้: ต้องตั้งค่า Google Apps Script (ขั้นตอนที่ 1) เพื่อให้ข้อมูลอยู่ใน Google Sheet แทน

1
ตั้งค่า Google Apps Script

Google Apps Script คือ backend ฟรีที่รับออเดอร์จากเว็บและเก็บไว้ใน Google Sheet ของคุณ

1.1 สร้าง Google Sheet

  1. ไปที่ sheets.google.com แล้วกด + Blank spreadsheet
  2. ตั้งชื่อ เช่น INeedPOSN Orders 2568
  3. Copy Spreadsheet ID จาก URL:
    docs.google.com/spreadsheets/d/THIS_IS_YOUR_ID/edit

1.2 สร้าง Apps Script

  1. ใน Google Sheet กด Extensions → Apps Script
  2. ลบ code เดิมทั้งหมดออก
  3. วาง code จากไฟล์ appscript.gs (ไฟล์ที่ให้ไว้)
  4. แก้บรรทัดบนสุด:
    const SS_ID = 'ใส่ Spreadsheet ID ที่ copy มา';
  5. กด 💾 Save (Ctrl+S)

1.3 Deploy เป็น Web App

  1. กด Deploy → New deployment
  2. กด ⚙ icon แล้วเลือก Web app
  3. ตั้งค่าดังนี้: 
    Execute as:  Me (your account)
Who has access: Anyone
  4. กด Deploy → อนุญาต permission ที่ขอ
  5. Copy Web app URL ที่ได้มา เช่น:
    https://script.google.com/macros/s/AKfycbxXXXXXX.../exec
✅ ตอนนี้คุณมี URL ของ Apps Script แล้ว เก็บไว้ใช้ในขั้นตอนถัดไป
2
แก้ไข Config ในไฟล์ HTML

เปิดไฟล์ index.html และ admin.html ด้วย Notepad, VS Code, หรือ text editor ใดก็ได้ แล้วแก้บรรทัดด้านบนสุดของ <script>

index.html — config block

const SCRIPT_URL       = 'https://script.google.com/macros/s/AKfycbxXXX/exec'; ← Apps Script URL
const PROMPTPAY_NUMBER = '081-234-5678';      ← เบอร์ PromptPay จริง
const PROMPTPAY_NAME   = 'ภวัต เศรษฐเสถียร';
const PRICE_FULL       = 990;
const PRICE_SALE       = 690;
const PRICE_AFTER      = 790;
const SALE_DEADLINE    = '30 เม.ย. 2568';
const LINE_BIO  = 'https://line.me/ti/g2/XXXXXX'; ← Link LINE กลุ่ม
const LINE_CHEM = 'https://line.me/ti/g2/XXXXXX';
const LINE_COM  = 'https://line.me/ti/g2/XXXXXX';
const IG_HANDLE = '@ineedposn';              ← IG จริง

admin.html — config block

const SCRIPT_URL   = 'https://script.google.com/macros/s/AKfycbxXXX/exec';
const ADMIN_USERS  = { admin: 'รหัสผ่านที่คุณต้องการ' }; ← เปลี่ยนรหัสผ่าน!
⚠️ อย่าลืมเปลี่ยนรหัสผ่าน admin จาก ineedposn2568 เป็นรหัสผ่านที่ปลอดภัยกว่า
3
Deploy บน Netlify (ฟรี)

Netlify เป็นบริการ hosting ฟรีที่ง่ายที่สุด แค่ลาก-วางโฟลเดอร์ก็เสร็จ ได้ URL สาธารณะทันที

  1. ไปที่ netlify.com แล้ว Sign up ด้วย Google หรือ GitHub (ฟรี)
  2. หลัง login จะเห็นหน้า Dashboard กด Add new site
  3. เลือก "Deploy manually" (ลากไฟล์)
  4. ลากโฟลเดอร์ ineedposn/ ทั้งโฟลเดอร์วางลงในช่อง Drop zone
  5. รอ 10–30 วินาที → เว็บขึ้นออนไลน์ทันที!

จะได้ URL แบบ:

🌐 https://whimsical-mochi-abc123.netlify.app

เปลี่ยนชื่อ subdomain

  1. กด Site configuration → General
  2. กด Change site name
  3. ตั้งชื่อเป็น ineedposn หรือ ineedposn-2568
  4. URL จะเปลี่ยนเป็น:
    🌐 https://ineedposn.netlify.app
💡 เมื่ออัปเดตไฟล์ HTML ในอนาคต ให้กด Deploys → Drag and drop แล้ววางโฟลเดอร์ใหม่ได้เลย ไม่ต้อง setup ใหม่

ทางเลือกอื่น: GitHub Pages (ฟรี แต่ยากกว่านิดหน่อย)

  1. สร้าง GitHub account ที่ github.com
  2. สร้าง repository ชื่อ ineedposn
  3. Upload ไฟล์ทั้งหมด
  4. ไปที่ Settings → Pages → Branch: main → Save
  5. URL: https://YOUR_USERNAME.github.io/ineedposn/
4
ทดสอบระบบ

หลังจาก deploy เสร็จ ทดสอบตามลำดับนี้:

  1. เปิดหน้าหลัก — ดูว่า index.html โหลดได้ ราคาแสดงถูกต้อง
  2. ทดสอบซื้อ — กดสมัครเรียน กรอกข้อมูล สร้าง username/password ดู ORDER ID และข้อมูลบัญชีที่ได้
  3. เปิด Admin — ไปที่ /admin.html login ด้วย username/password ที่ตั้งไว้ ดูว่าออเดอร์ขึ้นมาใน tab ออเดอร์
  4. สร้างบัญชีนักเรียน — ใน admin กด ✓ อนุมัติออเดอร์ → บัญชีนักเรียนจะเปลี่ยนสถานะ (หรือสร้างเองใน tab ผู้เรียน)
  5. ทดสอบ login นักเรียน — เปิด /learn.html login ด้วย username/password ที่สร้างไว้
  6. ใส่เนื้อหา — ใน admin tab "เนื้อหาคอร์ส" → ใส่ YouTube Video ID ของคลิปที่จะสอน
✅ ถ้าทุกขั้นตอนทำงานได้ เว็บพร้อมใช้งานจริงแล้ว!
⚠️ ปัญหาที่พบบ่อย: ถ้า Admin ไม่เห็นออเดอร์ → ตรวจสอบว่าได้ตั้งค่า Apps Script และใส่ URL ถูกต้องหรือยัง เพราะออเดอร์ถูกเก็บใน localStorage ของเบราว์เซอร์ที่สมัคร ไม่ใช่เบราว์เซอร์ของ Admin
5
เพิ่มรูปภาพ

QR PromptPay

  1. Save รูป QR PromptPay จริงของคุณเป็น qr-promptpay.png
  2. วางไว้ในโฟลเดอร์เดียวกับ index.html
  3. เปิด index.html ค้นหา qr-promptpay.png
  4. ลบ SVG ปลอมออก แล้ว uncomment บรรทัด <img src="qr-promptpay.png">
💡 วิธีสร้าง QR PromptPay: เปิด app ธนาคาร → รับเงิน → QR พร้อมเพย์ → screenshot หรือ save รูป

รูปผู้สอน

  1. เตรียมรูปหน้าตรงของแต่ละคน ขนาดแนะนำ: อย่างน้อย 300×300 px
  2. บันทึกเป็น photo-bio.jpg, photo-chem.jpg, photo-com.jpg
  3. วางในโฟลเดอร์เดียวกับ index.html
  4. เปิด index.html หา section Instructors แล้วเพิ่ม:
    <img src="photo-bio.jpg" alt="พี่พร้อม"
  style="width:80px;height:80px;border-radius:50%;object-fit:cover;margin-bottom:12px">
6
Custom Domain (ถ้ามี)

ถ้าต้องการ URL แบบ ineedposn.com แทนที่จะเป็น ineedposn.netlify.app

  1. ซื้อ domain จาก Namecheap หรือ domains.google.com ราคาประมาณ ฿300-500/ปี สำหรับ .com
  2. ใน Netlify → Site configuration → Domain management → Add custom domain
  3. กรอก domain ที่ซื้อมา เช่น ineedposn.com
  4. ทำตามขั้นตอน DNS verification ที่ Netlify บอก
  5. Netlify จะให้ SSL certificate (HTTPS) ฟรีอัตโนมัติ
💡 ยังไม่จำเป็นต้องมี custom domain ก็ได้ ใช้ ineedposn.netlify.app ไปก่อนได้เลย

คำถามที่พบบ่อย

ทำไม Admin ไม่เห็นออเดอร์?

เพราะ localStorage เก็บข้อมูลในเบราว์เซอร์เดียว ต้องตั้งค่า Google Apps Script ให้เสร็จก่อน แล้วแก้ SCRIPT_URL ในทั้ง index.html และ admin.html จากนั้น admin.html จะดึงข้อมูลจาก Google Sheet ได้

ข้อมูลนักเรียนอยู่ที่ไหน?

ปัจจุบัน: localStorage ของแต่ละเบราว์เซอร์ — เหมาะสำหรับ test และเริ่มต้น
ระยะยาว: เพิ่มระบบ sync กับ Google Sheet ผ่าน Apps Script

Netlify ฟรีมีข้อจำกัดอะไรบ้าง?

Netlify Free: Bandwidth 100GB/เดือน, Build minutes 300 min/เดือน — เพียงพอมากสำหรับเว็บ HTML static ขนาดนี้ ไม่ต้องกังวลเรื่องค่าใช้จ่าย

เพิ่มวิดีโอ YouTube อย่างไร?

เปิด Admin → tab เนื้อหาคอร์ส → เลือกวิชา → กด "+ บทเรียน" → ใส่ YouTube Video ID (11 ตัวอักษรหลัง ?v= ใน URL)

นักเรียนสมัครแล้วเข้าเรียนได้ทันทีไหม?

ไม่ได้ครับ ต้องรอ Admin อนุมัติก่อน Admin เปิด admin.html → tab ออเดอร์ → ดูสลิป → กด ✓ อนุมัติ → บัญชีนักเรียนจะเปลี่ยนสถานะเป็น approved → เข้าเรียนได้
